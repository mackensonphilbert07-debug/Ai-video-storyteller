## 🚀 Deployment Steps

### Step 1: Push Code to GitHub

```bash
# Navigate to your project directory
cd ai-video-storyteller

# Initialize Git (if not already done)
git init
git add .
git commit -m "Initial commit: AI Video Storyteller with full AI features"

# Add remote
git remote add origin https://github.com/YOUR_USERNAME/Ai-video-storyteller.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 2: Create Render Blueprint

1. Go to https://render.com/dashboard
2. Click "New +" → "Blueprint"
3. Connect your GitHub account
4. Select the `Ai-video-storyteller` repository
5. Render will automatically detect the `render.yaml` file

### Step 3: Configure Environment Variables

In the Render dashboard, set these environment variables:

**Required (from Manus):**
- `VITE_APP_ID` - Your Manus OAuth App ID
- `OWNER_OPEN_ID` - Your Manus Owner ID
- `OWNER_NAME` - Your name
- `BUILT_IN_FORGE_API_KEY` - Your Manus API Key
- `VITE_FRONTEND_FORGE_API_KEY` - Your Manus Frontend API Key
- `VITE_ANALYTICS_ENDPOINT` - Optional: Analytics endpoint
- `VITE_ANALYTICS_WEBSITE_ID` - Optional: Analytics website ID

**Auto-generated:**
- `JWT_SECRET` - Auto-generated by Render
- `DATABASE_URL` - Auto-generated from PostgreSQL service

### Step 4: Deploy

1. Click "Deploy" in the Render dashboard
2. Wait for the build to complete (5-10 minutes)
3. Once deployed, you'll get a URL like: `https://ai-video-app.onrender.com`

## 📋 What Gets Deployed

### Single Service Architecture
- **Service:** `ai-video-app` (Node.js)
- **Database:** `ai-video-db` (PostgreSQL)

### Build Process
1. Installs dependencies with pnpm
2. Compiles frontend with Vite → `dist/public/`
3. Compiles backend with esbuild → `dist/index.js`
4. Runs database migrations
5. Starts Express server

### File Structure After Build
```
dist/
├── public/              # Frontend static files (served by Express)
│   ├── index.html      # Main HTML
│   ├── assets/         # JS, CSS bundles
│   └── __manus__/      # Debug collector
└── index.js            # Backend server bundle
```

## 🔍 Troubleshooting

### Issue: "AI Video Storyteller API is running" (No UI)

**Cause:** Frontend files not being served

**Solution:**
1. Check build logs in Render dashboard
2. Ensure `dist/public/index.html` exists
3. Verify `serveStatic()` is being called in production mode

### Issue: Database Connection Error

**Cause:** DATABASE_URL not set or incorrect

**Solution:**
1. Check PostgreSQL service is running
2. Verify DATABASE_URL in environment variables
3. Check database migrations completed

### Issue: OAuth Login Not Working

**Cause:** VITE_APP_ID or OAUTH_SERVER_URL incorrect

**Solution:**
1. Verify VITE_APP_ID matches your Manus app
2. Check OAUTH_SERVER_URL is `https://api.manus.im`
3. Verify callback URL is registered in Manus dashboard
